# Happy Hour Finder - Frontend

React + TypeScript frontend for the Happy Hour Finder application.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open http://localhost:5173

## Features

- Chat interface for finding happy hours
- Location sharing via browser geolocation
- Input validation (1500 chars, 200 words max)
- Responsive design

## Development

Make sure the backend is running on http://localhost:3000 before starting the frontend.
